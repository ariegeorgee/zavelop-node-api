# zavelop-node-api
Zavelop Restful Api with Node js and MongoDB

## Getting Started

Ini adalah sample api menggunakan nodejs dan mongoDB

### Prerequisites

```
Node JS
MongoDB
Express JS
Nodemon
```

### Installing

```
npm install
```
and then
```
nodemon server.js
```
